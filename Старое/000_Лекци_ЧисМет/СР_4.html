Первый способ¶

Интерполяционный многочлен¶

Дана таблица координат Теперь подставляем известные точки в наше выражение, и получаезм строки системы линейных уравнений Решаем полученную систему уравнений, нужно найти a. Числа это Х и Y. Затем система уравнений решается и коэффициены подставляются в уравнение Полученное записывается как ответ

В способ¶

Интерполяционный многочлен Лагранжа¶

$$P(x)=L_1(x)+L_2(x)+...+L_n(x)$$$$L_k(x)=y_k* \prod\limits_{i = 1;i\neq k}^n \frac{(x-x_i)}{(x_k-x_i)}$$

$ (x_1,y_1);(x_2,y_2);...(x_n,y_n);$ - интерполяционные узлы

function L_1(x);
    function xX(x,i,j);return (x-X[j])/(X[i]-X[j]);end;
    Y[1]*xX(x,1,2)*xX(x,1,3)*xX(x,1,4)
end;
function L_k(x,k);
    rez=Y[k];
    for i=1:size(X)[2];
        if i!=k;
            rez=rez*(x-X[i])/(X[k]-X[i]);
        end;
    end;
    return rez;
end;
function L(x);L_k(x,1)+L_k(x,2)+L_k(x,3)+L_k(x,4);end;

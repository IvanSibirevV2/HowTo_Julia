using Pkg
Pkg.add("IJulia") - Установить
-Если стоят 2 и более версии Julia то процесс непрокатит
-Придется снести Pkg "IJulia" из той версии Julia из которой ставил.
Pkg.rm("IJulia") - Снести
-Удальти их из жесткого диска влоб.
-Придется снести все версии Julia
-Затем придется удалить ядро Julia из Jupiter (Это отдельное приключения)
-Затем можно ставить  в прямом порядке...
